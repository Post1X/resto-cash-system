# RCS (Resto Cash System)

Этот проект представляет собой кассовую систему, созданную специально для сети ресторанов. 
Бэкенд разработан на Node.js, фронтэнд на WPF (C#) и предоставляет удобный интерфейс для облегчения рабочего процесса сотрудников и интеграции с различными системами ресторанного бизнеса.

## Запуск

Для запуска проекта используйте следующие команды:

```bash
yarn && yarn start
